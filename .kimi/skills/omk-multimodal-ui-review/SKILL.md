## Multimodal UI Review

Use this when the task includes screenshots, mockups, browser captures, Playwright traces, or videos.

## Steps

1. Inspect the visual input first.
2. Identify layout, spacing, typography, color, component hierarchy, and interaction state.
3. Compare against existing implementation if available.
4. Propose minimal code changes.
5. Verify responsive and accessibility impact.
